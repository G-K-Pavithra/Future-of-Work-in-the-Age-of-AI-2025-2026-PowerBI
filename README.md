# Future-of-Work-in-the-Age-of-AI-2025-2026---PowerBI
📊 Overview
This project analyzes how Artificial Intelligence is transforming the workforce by examining automation risk, salary changes, skill gaps, and AI adoption across industries and countries.
🎯 Objectives
Analyze automation risk across industries
Study AI adoption globally
Evaluate salary changes by job roles
Identify skill gaps and reskilling needs
🛠 Tools Used
Excel – Data Cleaning
Power BI – Data Visualization & Dashboard
📁 Dataset
Source: Kaggle ( https://www.kaggle.com/datasets/algozee/future-of-work-in-the-age-of-ai-20202026 )
Years Used: 2025–2026
📈 Key Insights
High automation risk in industries like Finance and Energy
AI adoption is increasing globally
Salary impact varies by job role
Significant skill gaps exist across industries
High demand for reskilling
📊 Dashboard Features
Interactive filters (Year, Country, Industry)
KPI cards (Automation Risk, AI Adoption, Salary Change)
Visuals: Bar charts, Scatter plots, Map
📂 Files Included
Power BI Dashboard (.pbix)
Dataset (Excel)
Project Report (Word)
